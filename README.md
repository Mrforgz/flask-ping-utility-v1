# flask-ping-utility-v1

A lightweight, web-based utility for network availability monitoring. This tool allows users to perform ICMP pings via a web interface and maintains a persistent, structured audit log of all activity.

## Features
* **Web-Based Pinging:** Simple browser-based interface to check host reachability.
* **Structured Audit Trail:** Automatically logs every scan attempt with clear session delimiters.
* **Lightweight:** Built with Flask for minimal resource consumption.

## How to Run
1. Ensure you have Python installed.
2. Install Flask: `pip install flask`
3. Run the application: `python app.py`
4. Navigate to `http://127.0.0.1:5000` in your browser.

## Project Status
This is a starter project focused on understanding web-request lifecycles and file I/O operations. Future updates will include port scanning and enhanced diagnostic features.
